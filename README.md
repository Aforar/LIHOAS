# LIHOAS 🏨

**LIHOAS** is a luxury hotel platform in Agadir, Morocco — built with [Next.js](https://nextjs.org/), [next-intl](https://next-intl-docs.vercel.app/), and [Tailwind CSS](https://tailwindcss.com/).

## Features

- 🌍 Full trilingual support: **English**, **French**, **Arabic** (with RTL)
- 🏨 Multi-page: Home, About, Stays, Booking Form
- 🎨 Premium design with smooth scroll animations
- 📅 Client-side validated booking form
- 📍 Agadir local attractions section

## Tech Stack

- **Framework**: Next.js 16
- **Internationalization**: next-intl (EN / FR / AR)
- **Styling**: Tailwind CSS v4
- **Animations**: Framer Motion, Lenis smooth scroll
- **3D Background**: @shadergradient/react

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
